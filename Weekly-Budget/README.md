# Weekly Budget Tracker

The Weekly Budget Tracker is a simple web application that helps users track their weekly expenses. Users can provide their credentials, set their weekly budget, and add their expenses. The application displays the expenses in a tabular format and provides a pie chart visualization for better understanding.

## URL

https://beautiful-moxie-f53dc5.netlify.app


## Features

- User authentication: Users can register with their full name, email, and phone number to create an account.
- Set weekly budget: Users can set their weekly budget to manage their expenses effectively.
- Add expenses: Users can add their daily expenses, including the expense name and amount.
- View expenses: The application displays all expenses in a tabular format, showing the expense name, amount, and date.
- Visualize expenses: Expenses are visualized using a pie chart to provide a clear overview of spending categories.

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap 5

## How to Use

1. Clone the repository: `git clone <repository_url>`
2. Navigate to the project directory: `cd weekly-budget`
3. Open the `index.html` file in your web browser.
